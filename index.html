    Sobre o Genéricos BR


    • Genéricos BR 0.9 - Versão Web
      © 2011 Marcos Gurgel — tapush.com

      Genéricos BR é um guia de medicamentos de referência e seus equivalentes genéricos destinado tanto a profissionais de saúde brasileiros quanto ao público em geral. Esta é uma versão para a web do aplicativo desenvolvido originalmente para iPhone.

      A versão web do Genéricos BR foi desenvolvida especialmente para usuários de Android, webOS e outros smartphones.

      Se você tem um iPhone, iPad ou iPod Touch, baixe o aplicativo na App Store
